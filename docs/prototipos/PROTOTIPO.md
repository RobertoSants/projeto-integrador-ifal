# Documentação de UX/UI - Banco de Talentos

## Fluxos Mapeados (Baseado em ENDPOINTS.md)
1. **Cadastro (POST /accounts/register):** Foco em coletar Município e Estado logo no início.
2. **Busca Regionalizada (GET /search):** Priorização visual de quem possui `is_local: true`.
3. **Criação de Perfil (POST /workers):** Interface assistida por IA para preenchimento da `bio`.

## Elementos de Acessibilidade
- Tipografia legível (mínimo 16px para corpo de texto).
- Botões com área de toque mínima de 44x44px.
- Linguagem simplificada nos rótulos de formulário.
